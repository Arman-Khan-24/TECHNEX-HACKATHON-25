Introduction to JavaScript

JavaScript is a dynamic programming language that's used for web development, in web applications, for game development, and lots more.

JavaScript was created by Brendan Eich in 1995 during his time at Netscape Communications.

It's now one of the world's most popular programming languages and is a core technology of the World Wide Web.

JavaScript Syntax

Let's look at a simple JavaScript example:

// Your first JavaScript code
function greet() {
  console.log("Hello, World!");
}

greet();

JavaScript Output

JavaScript can display output in different ways:

// Writing to console
console.log("Hello World!");

// Writing to HTML
document.write("Hello World!");

// Writing to an HTML element
document.getElementById("demo").innerHTML = "Hello World!";

// Using window alert
alert("Hello World!");

JavaScript Comments

Comments can be used to explain JavaScript code and make it more readable.

// This is a single-line comment

JavaScript Variables

Variables are containers for storing data values. In JavaScript, we have three ways to declare variables:

var x = 5;       // Old way (avoid using var)
let y = 6;       // Block-scoped variable
const z = 7;     // Constant (cannot be reassigned)

JavaScript Data Types

JavaScript has several data types:

// Number
let length = 16;

// String
let color = "Yellow";

// Boolean
let isActive = true;

// Object
let person = {
  firstName: "John",
  lastName: "Doe"
};

// Array
let fruits = ["Apple", "Banana", "Orange"];

// Undefined
let undefined_var;

// Null
let empty = null;

JavaScript Operators

JavaScript includes various operators for performing operations:

// Arithmetic Operators
let sum = 5 + 2;    // Addition
let sub = 5 - 2;    // Subtraction
let mul = 5 * 2;    // Multiplication
let div = 5 / 2;    // Division
let mod = 5 % 2;    // Modulus

// Comparison Operators
let equal = 5 == "5";     // true
let strictEqual = 5 === "5";  // false

JavaScript Strings

Strings are used for storing and manipulating text:

let text = "Hello World";
let length = text.length;  // Get length
let upper = text.toUpperCase();  // Convert to upper case
let lower = text.toLowerCase();  // Convert to lower case

JavaScript Functions

Functions are blocks of code designed to perform a particular task:

// Function Declaration
function myFunction(p1, p2) {
  return p1 * p2;
}

// Arrow Function
const multiply = (a, b) => a * b;

// Function Expression
const greet = function(name) {
  return "Hello " + name;
};
